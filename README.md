Designed for use with RunUO 1.0 but can be used to relay text from any .txt file. 
<br>
Open Project.cs, add your custom Webhook URL and set your .txt file location.
<br>
<br>
<br>
**To build the .exe, run the following command from the project folder:**
<br>
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
